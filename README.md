# Metabolic Modeling E. coli on Acetate (FBA)
Mục đích: Mô hình hóa chuyển hóa và dự đoán tốc độ tăng trưởng tối ưu của E. coli K-12 trên môi trường Acetate sử dụng Flux Balance Analysis (FBA).

Yêu cầu (Dependencies):
Python 3.8
COBRApy (v0.29.1 hoặc mới hơn)
Solver: GLPK (đã được cài đặt và cấu hình)

Cách chạy:
Tải file ecoli_acetate_model_final.py.
Chạy bằng lệnh: python ecoli_acetate_model_final.py
